# 🎂 Smart Age Calculator

A responsive full stack web application built using Flask, HTML, CSS, JavaScript, and SQLite.  
This project allows users to securely sign up, log in, calculate their exact age instantly, and view detailed age statistics in an attractive dashboard.

---
## 🌐 Live Demo

https://smartagecalculator.onrender.com


## Features
- Login & Signup System
- Smart Age Calculation
- Birthday Countdown
- Dark Theme
- Responsive Dashboard
- Previous History Tracking
- Real-Time Seconds Counter
- SQLite Database Integration
